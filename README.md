# **🐱 About #OOTD**

### 당신이 오늘 무엇을 입어야 할지 고민되시나요?  

**#OOTD**는 선택 장애가 있는 사용자들을 위한 맞춤형 패션 큐레이터 앱입니다. 

재미있는 테스트를 통해 나만의 시그니처 룩을 찾아보세요. 

**당신의 취향에 딱 맞는 스타일 추천과 구매까지** 모두 가능합니다!

# **✨ 주요 기능**

 ✅ 패션 취향 테스트
   - 사용자의 패션 스타일을 분석하기 위한 다양한 테스트를 제공합니다.
   - 이를 통해 맞춤형 스타일과 **오늘의 OOTD(Outfit of the Day)**를 추천받을 수 있습니다.

 **💬 AI 채팅을 통한 스타일 제안 기능**
   - 어떤 장소에 가는지, 누구를 만나는지, 그리고 어떤 분위기를 원하는지에 따라 **티디**가 맞춤 스타일을 제안합니다.

 **👕 나만의 옷장 기능**
   - 기존에 자신이 가진 아이템들로 코디를 해볼 수 있습니다.
   - **SNS**처럼 자신만의 OOTD를 올리고, 공유할 수 있습니다.

 **📖 룩북 기능**
   - 내 취향에 맞는 다양한 스타일들을 볼 수 있고, 코디된 아이템들을 확인할 수 있습니다.

# **📱 주요 화면 구성**

## 3. **#OOTD**
   - 취향에 맞는 룩북이 보여지며 정보를 확인할 수 있습니다.
   - 오늘의 OOTD(Outfit of the Day)를 다른 사용자와 공유하며 비슷한 취향을 가진 사람들의 스타일을 확인하고 영감을 얻을 수 있습니다.

   ![ootdView](https://github.com/user-attachments/assets/9d4a0c33-e2db-4dba-9a2f-de1242b591c0)

## 4. **옷장**
   - **나의 옷장**
     - 내가 가지고 있는 아이템을 티디가 조합해서 스타일링해주고 화면에 보여줍니다.
     - 소유 중인 아이템의 목록을 보여주고 구매한 코디템을 추가할 수 있습니다.
   - **나의 OOTP**
     - #OOTD에 내가 오늘 입었던 코디를 공유할 수 있고, 내가 공유했던 OOTD를 확인할 수 있습니다.

## 🧑‍🏫 튜나 강사님의 프리미엄 피드백

### 테스트 화면

	•	테스트 설문: 설문지의 행간을 더 여유 있게 조정
	•	결과지 시각화 강화: 브랜드 추천을 시각적으로 더 강조
	•	사용자 이해도 개선:
	•	체형 설명이 부족한 사용자를 위해 사진이나 설명 추가
	•	성별 항목에 ‘기타’ 옵션 추가
	•	로그인 화면 개선 필요: 로그인 화면이 이탈률을 높일 수 있음

### 홈 화면

	•	검색 레이아웃 조정: 홈 화면에서 검색 레이아웃을 개선 필요
	•	키보드 이슈 해결: 검색 및 채팅 시 키보드가 안 내려가는 문제 해결
	•	스크롤뷰 조정: 스크롤뷰의 높이가 작다는 피드백
	•	상세 페이지 네비게이션 추가: 상품을 누르면 상세페이지에서 뒤로가기가 없어 나갈 수 없음

### OOTD 기능 개선

	•	컬렉션 뷰 간격 조정: OOTD 화면에서 컬렉션 뷰 간격을 맞추기
	•	SNS 기능 강화: 사람들의 허영심을 자극하는 기능이 필수
	•	네비게이션 추가:
	•	OOTD > OOTD 상세 > 제품 상세로 이어지는 네비게이션 필요
	•	채팅에서 테스트 페이지로 넘어가는 네비게이션 버튼 추가 제안

### 옷장 기능 개선

	•	컬렉션 뷰 추가 필요: 옷장 화면에서 컬렉션 뷰가 적음

### 추가 피드백

	•	이미지 파일 수정
	•	리드미 수정

 ### 🎬 시네마팀

	•	테스트를 ‘추천’ 탭으로 만들어 ‘날씨’ 등 외부 요인에 따라 데일리 코디를 추천하는 기능 추가
	•	로그인 후에도 테스트 가능하도록 설정 → 챗봇에 버튼 추가
